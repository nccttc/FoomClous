## 🌟 项目亮点

- **多端支持**：Web 端管理 + Telegram 机器人快捷上传。
- **存储自由**：原生支持 Local, S3, WebDAV, OneDrive, OSS。
- **极致体验**：支持视频在线预览、GIF 自动播放、断点续传。
- **轻量可靠**：基于 Node.js 与 React 构建，全自动化运维工具。
