## 🌟 项目亮点

- **多端上传**：支持 Web 后台管理与 Telegram 机器人一键转存。
- **广阔兼容**：原生适配本地磁盘、S3 兼容存储、WebDAV、OneDrive 等主流云端。
- **智能体验**：支持视频在线预览、GIF 动图自动播放及大文件断点续传。
- **隐私保护**：所有云端凭据仅本地保存，通过服务器代理模式避免密钥泄露。

## ⚠️ 现有发现 / 已知问题

### `/ytdlp` 画质挡位可能不生效

在解析部分视频链接时，即使在 `/ytdlp` 命令中选择了不同画质挡位（最高/标清/低画质），最终下载结果仍可能是**最高清画质**。

这会带来困扰：用户期望低画质文件更小，但实际文件大小和清晰度没有明显差异。

该问题通常与源站可用的格式/分辨率组合有关（例如某些站点只提供单一可用视频格式，或格式信息不完整导致自动选择结果一致）。后续会继续优化选择逻辑与兼容性。
